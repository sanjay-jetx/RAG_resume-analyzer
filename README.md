# 🚀 AI Resume Analyzer (RAG Chatbot)

An AI-powered Resume Analyzer built using **Retrieval-Augmented Generation (RAG)**.  
Upload multiple resumes (PDF) and ask questions to get intelligent insights about candidates.

---

## 📌 Features

- 📄 Upload multiple PDF resumes
- 🔍 Ask questions about candidates
- 🧠 AI-powered answers using RAG
- 🏆 Find best candidate for a role
- 🎯 Optional candidate name filtering
- ⚡ Fast retrieval using FAISS vector database

---

## 🛠 Tech Stack

- Python 🐍
- Streamlit 🎨
- LangChain 🔗
- FAISS 🧠
- OpenAI API 🤖
